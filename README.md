# 🎓 Cadastro de Alunos

### Projeto desenvolvido em **Java** para cadastro de alunos, cálculo de média e classificação acadêmica.

</div>

---

# #📖 Sobre o projeto

Este projeto foi desenvolvido para praticar os conceitos fundamentais da linguagem **Java**, utilizando entrada de dados via teclado, métodos, estruturas condicionais e cálculos matemáticos.

A aplicação realiza o cadastro de um aluno, calcula sua média final e informa automaticamente sua situação acadêmica de acordo com a média obtida.

---

# ✨ Funcionalidades

✔ Cadastro do aluno

✔ Entrada de três notas

✔ Cálculo da média final

✔ Classificação automática do aluno

✔ Exibição organizada das informações

# 📋 Regras de Negócio

## Média Final

A média é calculada através da fórmula:

```java
(nota1 + nota2 + nota3) / 3
```

---

## Situação do aluno

| Média         | Situação                   |
| ------------- | -------------------------- |
| 9,0 ou mais   | 🌟 Aprovado com Excelência |
| 7,0 a 8,9     | ✅ Aprovado                 |
| 5,0 a 6,9     | 📚 Recuperação             |
| Abaixo de 5,0 | ❌ Reprovado                |

# 🚀 Como executar

1. Clone o repositório:

```bash
git clone https://github.com/SEU_USUARIO/java-cadastro-aluno.git
```

2. Entre na pasta do projeto:

```bash
cd java-cadastro-aluno
```

3. Abra o projeto no IntelliJ IDEA.

4. Execute a classe

```bash
java CadastroAlunos
```
